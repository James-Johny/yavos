# Lista de Colaboradores

Este projeto é uma aplicação web simples para gerenciar e visualizar uma **lista de colaboradores**.  
Ele foi desenvolvido utilizando **HTML, CSS e JavaScript**, com suporte a **PWA (Progressive Web App)** para funcionamento offline.

---

## 📂 Estrutura do Projeto

- `index.html` → Página principal da aplicação.
- `estilos.css` → Folha de estilos para layout e design.
- `script.js` → Lógica principal em JavaScript.
- `sw.js` → Service Worker para cache e suporte offline.
- `manifest.json` → Configuração do PWA (nome, ícones, comportamento).
- `icon-192.png` / `icon-512.png` → Ícones do aplicativo.
- `listaepis.csv` → Base de dados em formato CSV.
- `bancodehoras.pdf` → Documento auxiliar com informações de banco de horas.

---

## 🚀 Funcionalidades

- Visualização de colaboradores a partir de um arquivo CSV.
- Interface responsiva e estilizada.
- Suporte a instalação como **aplicativo PWA**.
- Funcionamento offline via Service Worker.
- Documentos auxiliares integrados.
